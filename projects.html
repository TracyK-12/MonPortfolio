EMS

Application de gestion de stock

2024 Projet Personnel

EMS est une application web complète de gestion de stock développée en PHP. Elle permet une gestion intuitive des articles, catégories, utilisateurs et rapports détaillés. L'application intègre un système de contrôle d'accès robuste et un dashboard administrateur puissant pour la surveillance en temps réel.

Fonctionnalités

  • Gestion complète des stocks et articles
  • Système d'utilisateurs avec rôles
  • Rapports et statistiques
  • Dashboard administrateur
  • Contrôle d'accès granulaire

Quiz Time

Application de quiz interactive

2024 Projet Académique

Quiz Time est une application React moderne conçue pour offrir une expérience de quiz engageante et interactive. Avec un système de scoring en temps réel, des timers et une intégration API REST, elle combine performance et design réactif pour captiver les utilisateurs.

Fonctionnalités

  • Questions dynamiques via API REST
  • Système de scoring en temps réel
  • Timer pour chaque question
  • Interface réactive et responsive
  • Historique et statistiques
